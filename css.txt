2. Типы позиционирования CSS
   static - дефолтный тип позиционирования 
   relative - элемент  позиционируется относительно своего текущего положения
   absolute -позиционирование относительно другого элемента у которого позиционирование не static. если такого нет,
   то относительно  окна браузера
   fixed - позиционирование только относительно окна браузера
   sticky - в видимой области видит себя как fixed, при дальнейшей прокрутке, скроллится вместе с родителем


3. Что такое селектор? И какие селекторы существуют?
   селектор это часть css правила, которая сообщает браузеру к какому элементу или элементам веб страницы будет применен стиль
   Есть простые и составные селекторы
   простые это тип в котором указывается один вариант селектора
   Составной селектор это комбинация селекторов
